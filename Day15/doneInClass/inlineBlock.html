Inline Block goodness

Inline-block is a value for the display property that lets you consistently apply properties like margins, padding, width, height, and vertical-align while remaining an inline element. Itâ€™s incredibly useful in any situation where you want to align block elements horizontally in rows.

A mainstay of blogs everywhere is the use of an unordered list for the navigation menu. Turning that list horizontal would normally require you to float all the menu items left or right, then deal with the height of the containing element, and any clearing problems. All of these problems are nullified by applying display: inline-block; to all the li elements. If you want to make the anchor elements fill up the space, apply display: block; to them.

Inline-block features

  • inline-block renders the white space in your html, because it treats the elements as characters
  • support is excellent: IE8+ supports it
  • it is easy to align elements vertically among each others: use text-align on the parent element
